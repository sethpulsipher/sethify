# Media Requirements

## Images

| Asset | Location | Dimensions | Format | Notes |
|---|---|---|---|---|
| Hero background | Home hero | 1920×1080 | WebP (JPG fallback) | High contrast for text overlay. Editorial feel. Grain texture overlay applied via CSS, not baked into image. |
| About photo | About intro | 800×600 | WebP (JPG fallback) | Founder photo or workspace shot. Keep authentic, not stock. |
| Industry cards | Home "Who We Work With" | 600×400 | WebP | One per industry vertical (HVAC, dental, legal, plumbing, contractors). Can be icons or photos. |
| Service images | Service pages | 800×600 | WebP | One hero/header image per service page (web design, AI systems, Shopify). |
| OG image | Meta tags (all pages) | 1200×630 | JPG | Sethify logo on cream (#FDF8F3) background. Used for social sharing. |
| Favicon | Browser tab | 32×32 + 180×180 | ICO + PNG | Icon only. No wordmark. Coral (#FF6B4A) on transparent or ink (#0F172A) background. |
| Blog post thumbnails | Blog index + OG | 1200×630 | WebP | One per blog post. Consistent template style. |

## Videos

No video assets required at launch. Add when case studies or demos are available.

## Icons

- **Icon library:** Lucide React (`lucide-react`)
- **Style:** Outline (default Lucide style)
- **Size:** 24px standard, 20px inline with text
- **Color:** Inherit from parent text color. Use coral (#FF6B4A) for accent icons only.

## Logo

- **Primary:** SVG (for web). Ink (#0F172A) on transparent.
- **Alternate:** SVG. Cream (#FDF8F3) on transparent (for dark/ink backgrounds).
- **PNG fallback:** Export at 2x for retina. Both colorways.
- **Favicon:** Derived from logo mark (icon only, no wordmark).
